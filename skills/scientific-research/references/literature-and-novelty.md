# Literature And Novelty

Use this reference for literature reviews, novelty checks, related work, citation grounding, and evidence briefs.

## Search Protocol

1. Translate the research question into search concepts.
2. Build search strings with synonyms, field-specific terms, and likely method names.
3. Search multiple source types when available:
   - Peer-reviewed papers and preprints.
   - Official datasets, standards, clinical guidelines, government or institutional reports.
   - Reputable technical documentation for software or methods.
4. Track inclusion and exclusion criteria.
5. Record the search date when currency matters.

## Source Quality Tiers

- Tier 1: Systematic reviews, meta-analyses, major benchmarks, standards, official statistics, validated datasets.
- Tier 2: Peer-reviewed studies, respected conference papers, technical reports with methods and data.
- Tier 3: Preprints, working papers, documentation, credible expert commentary.
- Tier 4: Blogs, news, marketing pages, unsourced summaries. Use only for context unless the claim is about the page itself.

## Evidence Ledger Fields

Use these columns:

```text
Source | Date | Claim supported | Evidence type | Quality tier | Limitations | Notes
```

For source-backed answers, cite exact URLs, DOIs, titles, or user-provided file paths. Do not fabricate citation metadata.

## Novelty Matrix

Compare the proposed work against known work:

```text
Prior work | Similarity | Difference | Evidence gap | Novelty risk | How to verify
```

Novelty claims should be conservative:

- "I did not find evidence of X in the searched sources" is acceptable.
- "No one has ever done X" requires strong evidence and is usually too broad.

## Citation Integrity

Before writing:

- Check that every cited source supports the sentence it is attached to.
- Avoid citation laundering, where a review cites another source that may not support the claim.
- Avoid overclaiming from small samples, narrow populations, synthetic data, or benchmark-only results.
- Mark uncited background as assumption or common framing, not as established fact.

## When Web Access Is Unavailable

Proceed with:

- Search strategy.
- Expected source classes.
- Evidence gaps.
- Provisional hypotheses.

Clearly state that literature currency and citations still need verification.
